# Senyal — Landing Page

Static landing page for Senyal ("Deja de perder dinero con los plantones").

Single self-contained `index.html` bundle — all assets are embedded.

## Deploy

Point Vercel (or any static host) at the repo root. `index.html` is served directly, no build step required.

```bash
# Local preview
python3 -m http.server 8000
# → http://localhost:8000
```
